# Studi Kasus Point of Sales - Versi Laravel

## Running

- cd pos-laravel
- cp .env.example .env
- ./vendor/bin/sail up -d
- ./vendor/bin/sail artisan migrate
- ./vendor/bin/sail artisan db:seed
- visit <http://localhost>

## Test with Postman

- Silahkan cek di modul atau materi **MENGUJI DENGAN POSTMAN**
